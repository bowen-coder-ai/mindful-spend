# Minimal Web Accounting App

A minimalist accounting application with AI-powered transaction parsing.

## Setup

1.  **Install Dependencies**
    Since the agent environment could not access `npm`, please run:
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Get a Gemini API Key**
    - Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
    - Create a free API key.
    - Click the "Settings" icon in the app to paste your key.

## Features

- **Voice/Text Input**: "Lunch $20"
- **AI Parsing**: Automatically detects Amount, Category, and Date.
- **Visuals**: Morandi color palette, smooth animations.
- **Privacy**: Data stored locally in your browser. API Key stored locally.
